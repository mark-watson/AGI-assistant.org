AGI Assistant Overview

Intelligense and knowledge are a form of compression. The AGI Assistant uses text as memory. These texts are comprised of stored conversations with users, reference texts used to answer user's previous queries, and any reference text provided to the AGI Assistant.

For efficiency all memory texts are both summarized and indexed using vector embeddings,

Standing on the shoulders of giants

The tools used to build the AGI Assistant are: LangChain, LlamaIndex, the OpenAI APIs, and open models from Hugging Face.
